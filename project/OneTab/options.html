OneTab Options
Restrict URL
URLs If a new URL is in options configured URLs.
The new URL will open in the already existing tab.

Rules: 
1. One url per line.
2. Url with https:// or http:// prefix.

Examples: 
https://youdao.com/
https://translate.google.com/
Notification Show notification when OneTab triggered on URL.
Extract URL
URL RegExps If the input to address bar is matched by regexps.
The tab will open the url matched.

Rules: 
1. One url regexp per line.

Examples: 
(http%3A%2F%2Fxhslink\.com%2F[0-9|a-z|A-Z]{6})%EF%BC%8C
(https%3A%2F%2Fb23\.tv%2F[0-9|a-z|A-Z]{7})
(https%3A%2F%2Fv\.douyin\.com%2F[0-9|a-z|A-Z]{1,})%2F
